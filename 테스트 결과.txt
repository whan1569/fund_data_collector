# 데이터 수집 모듈 테스트 결과 (2024-03-16)

## 1. fetch_stocks.py (Yahoo Finance API)
- 상태: ✅ 정상 작동
- 수집 데이터: 주요 지수 (S&P 500, 다우존스, 나스닥, FTSE, 닛케이)
- 저장 위치: fund_bot/data/stocks.parquet
- 특이사항: 없음

## 2. fetch_commodities.py (Yahoo Finance API)
- 상태: ✅ 정상 작동
- 수집 데이터: 원자재 ETF (GLD, USO, SLV, CPER, JJN, PALL, JO, CORN, WEAT)
- 저장 위치: fund_bot/data/commodities.parquet
- 특이사항: Quandl API에서 Yahoo Finance API로 변경하여 ETF 데이터로 대체

## 3. fetch_bonds.py (FRED API)
- 상태: ✅ 정상 작동
- 수집 데이터: 국채 및 회사채 금리
- 저장 위치: fund_bot/data/bonds.parquet
- 특이사항: FRED API 키 정상 작동 확인

## 4. fetch_forex.py (Yahoo Finance)
- 상태: ✅ 정상 작동
- 수집 데이터: 주요 통화쌍 환율 (EUR/USD, USD/JPY, GBP/USD, USD/CHF, AUD/USD)
- 저장 위치: fund_bot/data/forex.parquet
- 특이사항: Forex-Python API에서 Yahoo Finance API로 변경하여 안정적으로 데이터 수집 가능

## 5. fetch_crypto.py (Binance API)
- 상태: ✅ 정상 작동

- 저장 위치: fund_bot/data/crypto.parquet
- 특이사항: Binance API 키 정상 작동 확인

## 6. fetch_real_estate.py (Yahoo Finance API)
- 상태: ✅ 정상 작동
